Model Architecture

ResNet18, fine-tuned
for binary classification

The model is a pretrained ResNet18 (ImageNet weights) with its final fully-connected layer replaced by a 2-class linear layer — one for Parasitized, one for Uninfected.

Images are resized to 224×224 and normalized using ImageNet statistics. Training used data augmentation (random flips, rotations) to improve generalization.

Training was done using Adam optimizer with cross-entropy loss over 10 epochs on the NIH Malaria Cell Images dataset, with the best checkpoint saved based on validation accuracy.

Dataset

NIH Cell Images Dataset — 27,558 images split 70/15/15 into train, validation, and test sets. Equal balance of Parasitized and Uninfected cells.
